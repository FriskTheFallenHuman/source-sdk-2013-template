Welcome to the SDK Template!<br>
<br>
<u>For Programmers:</u><br>
Check out "sdk_shareddefs.h" Inside this header, you will find a number of defines which activate functionality. By default, all of these defines are commented out.<br>
Features present in this template are:<br>
<b>*</b> Team System, complete with a team selection menu.<br>
<b>*</b> Player Class System, based heavily on proven Day of Defeat: Source, and Team Fortress 2 code. <br>
 - Simply define the classes in "sdk_shareddefs.cpp" and edit existing or create new script files in the scripts folder.<br>
<b>*</b> Prone and Sprinting functionality<br>
<b>*</b> Updated scoreboard functionality for teams and including Orange Box's new community avatar support<br>
<b>*</b> and more!<br>
<br>
<br>
<u>For Artists:</u><br>
<b>*</b> There is new model sources for the player included in this sample.<br>
<b>*</b> The sources for the sample map "sdk_teams_hdr" is also included; most tonemap controller inputs now function in multiplayer!<br>
